## 📘 一、用 Streamlit 做「點餐機」

### 🍔 點餐機在做什麼？

👉 讓使用者：

1. 輸入餐點
2. 按「加入」放進購物籃
3. 可以把不要的餐點刪掉

---

### 🧠 重要概念

#### ✅ session_state（記憶功能）

- `session_state` 就像 **電腦的記憶**
- 可以記住你點了哪些餐點
- 不會因為畫面重整就不見

📦 `orders` = 存放餐點的清單

---

#### ✅ 欄位（columns）

- 把畫面分成左右兩邊
- 左邊輸入餐點，右邊放按鈕

---

#### ✅ rerun（重新整理畫面）

- 加入或刪除餐點後
- 讓畫面立刻更新

---

## 📘 二、快速計算的「+=」運算

### ➕ 這些是「偷懶寫法」

| 寫法      | 意思      |
| --------- | --------- |
| `a += b`  | a = a + b |
| `a -= b`  | a = a - b |
| `a *= b`  | a = a × b |
| `a /= b`  | a = a ÷ b |
| `a //= b` | 只取整數  |
| `a %= b`  | 取餘數    |
| `a **= b` | 次方      |

👉 用來讓數學計算更快！

## 📘 三、while 迴圈（一直做，直到停）

### 🔁 while 是什麼？

👉 **只要條件是對的，就一直做**

#### 例子

- i 從 0 一直加到 10000
- 每次做完就 +1

---

### ⛔ break（強制停止）

- 遇到特別情況
- 立刻跳出迴圈

---

## 📘 四、for 迴圈（做固定次數）

### 🔢 for 在做什麼？

👉 **照次數一個一個來**

例子：

- 從 1 做到 5
- 每個數字跑一次

---

## 📘 五、random（亂數）

### 🎲 亂數是什麼？

👉 電腦幫你 **隨機選數字**

| 方法      | 說明       |
| --------- | ---------- |
| randrange | 範圍內隨機 |
| randint   | 包含頭尾   |

🎯 常用來：

- 骰子
- 抽獎
- 遊戲

---

## 📘 六、猜數字遊戲（進階）

### 🎮 遊戲規則

1. 電腦先想一個數字
2. 玩家猜
3. 電腦提示「大一點 / 小一點」
4. 猜對就贏 🎉

---

### 🧠 重點技巧

- 記住目標數字（session_state）
- 記住最小和最大範圍
- 猜對就重新開始

---

## 📘 七、字典（Dictionary）

### 📦 字典是什麼？

👉 像「對照表」

例子：

- 科目 → 成績
- 英文 → 中文

---

### 🔑 基本用法

- 看資料：`字典[鑰匙]`
- 新增資料
- 修改資料
- 刪除資料

---

### 🔍 常見功能

| 功能     | 說明           |
| -------- | -------------- |
| keys()   | 看所有鑰匙     |
| values() | 看所有值       |
| items()  | 鑰匙＋值       |
| pop()    | 刪資料         |
| in       | 有沒有這個東西 |
| len()    | 有幾筆資料     |

---

## 📘 八、成績平均計算（實用）

### 📊 在做什麼？

1. 每一科有 10 個成績
2. 把成績全部加起來
3. 除以 10
4. 算出平均分數

👉 很像老師算全班平均 👍
